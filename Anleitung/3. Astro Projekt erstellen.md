Astro Projekt erstellen mit Git Bash:

```
npm create astro@latest
```

Antworten während Installation:

```
Project name: obsidian-site
Template: minimal
Install dependencies: yes
Initialize git: yes
```

Danach:

```
cd obsidian-site
npm run dev
```

Lokale Website:

```
http://localhost:4321
```
