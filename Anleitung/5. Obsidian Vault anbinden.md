Empfohlen wird ein Symlink, damit neue Dateien automatisch übernommen werden.

## Windows

```
mklink /D src\content\notes C:\Users\Name\ObsidianVault
```

---

## Linux / Mac

```
ln -s ~/ObsidianVault src/content/notes
```

---

# Beispiel Obsidian Struktur

```
ObsidianVault/
├── Notiz1.md
├── Notiz2.md
├── Wissen/
│   └── AI.md
```

Website URLs:

```
/notes/notiz1
/notes/notiz2
/notes/wissen/ai
```
